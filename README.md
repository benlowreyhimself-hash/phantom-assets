# Phantom Assets Property Group

Static website for Phantom Assets Property Group - Property Finance & Development Excellence.

## Deployment

This site is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.

## Local Development

To serve locally, you can use any static file server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.
